# conceptual-kv

광고 키비주얼(KV)을 **컨셉 발산 → 카피 위임 → 무드별 타이포 아트디렉팅 → 실제 이미지 생성**까지 한 번에 처리하는 Claude Code 스킬.

실제 금융권 비딩 제안서(하나은행) KV 작업을 여러 번 반려·재작업하며 얻은 원칙을 스킬로 정리한 것입니다:
> "레퍼런스 그대로 베끼지 말 것", "카피가 PPT 텍스트처럼 밋밋하면 안 됨", "시안은 최소 6개", "비율 왜곡 금지"

## 이 스킬이 하는 것

| 단계 | 내용 |
|---|---|
| 1. 브리프 | 브랜드/제품/타겟/핵심카피 확인. 카피가 없으면 별도 카피 스킬로 위임 |
| 2. 컨셉 발산 | 오브제 치환·강제원근·타이포 개입·위트 코드 레버로 **최소 10개** 비주얼 컨셉 발산 (레퍼런스 모방 금지) |
| 3. 타이포 아트디렉팅 | 무드(다크/위트/감성/조형/미니멀/신뢰) → 서체 처리(크롬·붓캘리·마커·3D압출 등) 매핑 표로 프롬프트에 구체 지정 |
| 4. 실사 미장센 | 렌즈·조명·필름그레인·컬러그레이딩 지정으로 스톡틱 회피 |
| 5. 생성 | 이미지 생성 도구로 **최소 6개 시안** + 시안마다 컨셉/느낌/참고 한 줄 정리 |
| 6. 리뷰 게이트 | 벤치마크 밀도 계승 여부, 로고 고정, 비율 왜곡 여부 자체 점검 |

## 왜 만들었나

기존에 찾아본 광고/디자인 스킬들(canvas-design, ai-ads-claude 등)을 실제로 열어보니:
- `canvas-design`은 미술관/매거진 톤 지향 — 광고 임팩트와는 반대 방향(절제·저텍스트)
- `ai-ads-claude`는 카피·브리프는 잘 뽑지만 **텍스트 문서**만 출력 — 실제 이미지 생성 없음
- 이미지 생성 스킬들은 타이포그래피 아트디렉팅 기능이 약함

즉 "컨셉 발산 + 카피를 타이포로 디자인 + 실제 광고 톤 이미지"를 **한 파이프라인**으로 묶은 스킬이 없어서 직접 만들었습니다.

## 설치

### Claude Code
```bash
git clone https://github.com/koo950101/conceptual-kv-skill.git
mkdir -p ~/.claude/skills/conceptual-kv
cp conceptual-kv-skill/conceptual-kv/SKILL.md ~/.claude/skills/conceptual-kv/
```

### Claude.ai (웹)
`conceptual-kv/SKILL.md` 파일을 다운받아 Settings → Skills → "+"로 업로드.

## 사용 예

```
"이 카피로 KV 만들어줘: '하나만 믿고 가'"
"컨셉추얼하게 광고 이미지 하나 뽑아줘"
```

## 의존(선택)

카피를 새로 짓는 기능은 포함하지 않습니다. 카피가 없으면:
- 자체 카피 스킬이 있으면 그걸로 먼저 짓고 이 스킬에 넘기세요.
- 없으면 Claude에게 "이 제품에 맞는 후킹 카피 먼저 지어줘"라고 요청한 뒤 이 스킬을 이어서 쓰면 됩니다.
